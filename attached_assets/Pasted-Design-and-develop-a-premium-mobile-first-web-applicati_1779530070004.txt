Design and develop a premium mobile-first web application UI/UX with a sophisticated, minimal, modern aesthetic inspired by soft emerald/mint palettes and clean productivity dashboards.

The goal is NOT a flashy startup landing page.
The goal IS a polished, calming, high-trust interface system that feels intelligent, lightweight, and elegant.

=========================================
CORE UI/UX DIRECTION
=========================================

Create a highly refined interface system using:
- Soft mint greens
- Emerald gradients
- Deep evergreen dark tones
- Frosted glass surfaces
- Spacious layouts
- Smooth microinteractions
- Minimal visual clutter
- Premium typography hierarchy
- Consistent rounded corners
- Soft shadows only where necessary

The design should feel:
- calm
- modern
- intelligent
- professional
- interactive
- clean
- trustworthy
- mobile-native

Avoid:
- overly vibrant neon colors
- heavy skeuomorphism
- cluttered dashboards
- excessive shadows
- childish UI
- bulky cards
- crowded spacing
- overly animated interfaces
