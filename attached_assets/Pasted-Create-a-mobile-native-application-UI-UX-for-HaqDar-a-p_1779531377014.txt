This is NOT a desktop website.
This is a **real mobile app experience** designed primarily for:

* Android smartphones,
* one-handed usage,
* offline-first environments,
* vulnerable users under stress,
* discreet and privacy-sensitive interaction.

The app should feel like:

* a secure encrypted vault,
* a trauma-informed safety tool,
* a modern legal preparedness assistant,
* a discreet forensic diary.

Design for:

* React Native / Flutter style mobile UI,
* gesture-based navigation,
* native mobile interactions,
* biometric flows,
* bottom sheets,
* floating action buttons,
* swipe gestures,
* mobile transitions,
* touch-first layouts.
